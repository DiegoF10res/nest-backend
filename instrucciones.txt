comandos:

1. npm install @prisma/client

2. npm install --save-dev prisma

3. Se crea .env

4. npx prisma migrate dev --name init

5. npx prisma generate

6. npm install --save-dev cross-env

7. Se crea .env.test

8. npx prisma migrate dev --name init --env-file .env.test

9. Se ejecutan las pruebas: npm run test:e2e
